# doc2knowledge

A project to convert documentation into a knowledge base.

## Quickstart

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/doc2knowledge.git
   cd doc2knowledge
   ```

2. **Install dependencies**
   ```bash
   make install
   ```

3. **Run tests**
   ```bash
   make test
   ```

5. **Run linters**
   ```bash
   make lint
   ```
